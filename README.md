# IS345-Lab02-Registration-Form
This lab is part of the IS345 – Web Application Development 2 course and was implemented strictly for educational purposes. The objective of this lab is to practice creating a simple registration form using ASP.NET MVC and handling form data submission from the view to the controller.

using the debugger to detect and fix programming errors. Additional enhancements include modifying the form to add new fields and validating the submitted data during execution.

All data and examples used in this lab are fictional and are intended only for practicing the learned concepts. Any incorrect or incomplete information does not represent real-world applications.

# Technologies Used:
ASP.NET MVC
Razor Views
C#
HTML
Visual Studio Debugger

# Learning Outcomes
Creating and handling HTML forms in ASP.NET MVC

Passing data from views to controller actions

Using POST actions to process user input

Debugging and identifying logical and runtime errors
